코드 실행하는 방법
pip install -r requirements.txt 터미널에 위 명령어 실행 후
python app.py 실행

조영채 : 역할 - 깃허브 관리 및 AI 모델 적용, UI 최종수정 
주요 기여 내용 - 스펙트로그램을 AI가 분석하여 이미지를 생성하도록 코드 작성, 대본작성 및 발표
                사이트가 스크롤 없이 한 화면에 담기도록 UI개선, 다른 팀원들에게 깃허브의 사용법을 알려주며, PR된 코드들을 머지함.
 
이가현 : 역할 - 오디오 처리 코딩
주요 기여 내용 - 음성파일을 스펙트로그램(오디오시각화)으로 변환하는 코드 작성, ppt 제작

류태웅 : 역할 - 웹 프론트엔드 코딩
주요 기여 내용 - 웹 프론트엔드 구상 및 구현, 사용자의 경험에 기반하여 UI최적화
